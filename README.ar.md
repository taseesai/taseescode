**تأسيس كود** هو أداة CLI تعمل في الطرفية (Terminal) وتساعدك في كتابة الكود، مراجعته، تصحيحه، وبناء مشاريع كاملة — كل هذا بالعربي أو الإنجليزي. يدعم أكثر من 13 نموذج ذكاء اصطناعي، من النماذج المجانية إلى الأقوى في السوق.

صُنع بواسطة [TaseesAI](https://taseesai.com) في جدة، المملكة العربية السعودية.

## ما هو تأسيس كود؟

تأسيس كود أداة سطر أوامر (CLI) تضع مساعد ذكاء اصطناعي مباشرة في الطرفية. تكتب له بالعربي أو الإنجليزي، ويقدر:

- **يقرأ ملفاتك** ويفهم المشروع بالكامل
- **يكتب ويعدّل الكود** مباشرة في ملفاتك
- **ينفّذ أوامر** في الطرفية (npm install, git commit, إلخ)
- **يبحث في الكود** ويلقى الملفات اللي تحتاجها
- **يحلل الصور والفيديو** باستخدام نماذج الرؤية
- **يتذكر مشروعك** عبر ملف TASEESCODE.md

الفرق عن الأدوات الثانية؟ **تأسيس كود عربي أولاً**. مو ترجمة — مبني من الأساس ليفهم العربية ويتعامل معها بشكل طبيعي. تقدر تكتب "اعمل لي API بالنود" ويفهمك مباشرة.

### ليش تأسيس كود؟

- **١٣+ نموذج** — من المجاني (Llama, Qwen) إلى الأقوى (Claude, GPT-4o)
- **عربي أولاً** — يفهم العربي السعودي والفصحى
- **أسعار بالريال** — تتبع التكلفة بالريال السعودي
- **نظام مهارات** — وسّع قدراته بمهارات جاهزة أو مخصصة
- **ربط أي API** — Ollama, LM Studio, أو أي خدمة متوافقة مع OpenAI
- **مفتوح المصدر** — رخصة MIT

## الأدوات — ماذا يستطيع تأسيس كود أن يفعل

تأسيس كود يملك ٩ أدوات يستخدمها تلقائياً حسب طلبك:

| الأداة | الوصف |
|--------|-------|
| `read_file` | قراءة محتوى ملف |
| `write_file` | الكتابة في ملف موجود |
| `create_file` | إنشاء ملف جديد |
| `delete_file` | حذف ملف |
| `list_files` | عرض قائمة الملفات في مجلد |
| `run_command` | تنفيذ أمر في الطرفية |
| `search_code` | البحث في الكود |
| `git_diff` | عرض التغييرات في Git |
| `git_commit` | عمل commit في Git |

### أمثلة عملية

```
❯ اقرأ ملف package.json وقل لي ايش الاعتمادات
🔧 read_file: package.json
📋 الاعتمادات الموجودة: express, mongoose, jsonwebtoken...

❯ أنشئ ملف .env فيه إعدادات قاعدة البيانات
🔧 create_file: .env
✔ تم إنشاء .env بإعدادات MongoDB الافتراضية

❯ شغّل الاختبارات
🔧 run_command: npm test
✔ جميع الاختبارات نجحت (١٢/١٢)

❯ ايش التغييرات اللي ما سويت لها commit؟
🔧 git_diff
📋 ٣ ملفات معدلة: src/auth.ts, src/routes.ts, package.json
```

## الرؤية والصور

تأسيس كود يقدر يحلل الصور. أرسل مسار صورة أو رابط URL وسيتحول تلقائياً لنموذج يدعم الرؤية.

### النماذج التي تدعم الرؤية

- **Claude Sonnet** — أفضل تحليل للصور
- **GPT-4o** — تحليل متعدد الوسائط
- **Llama 3.2 Vision** — مجاني عبر Groq

### طريقة الاستخدام

```
❯ حلل لي هالصورة /path/to/screenshot.png
🔄 تم التحويل تلقائياً إلى: Claude Sonnet (يدعم الرؤية)
📸 تحليل الصورة...

الصورة تعرض واجهة تسجيل دخول فيها:
- حقل البريد الإلكتروني
- حقل كلمة المرور
- زر "تسجيل الدخول"
- رابط "نسيت كلمة المرور"

❯ حلل هالصورة https://example.com/design.png
🔄 تم التحويل تلقائياً إلى: GPT-4o (يدعم الرؤية)
📸 جاري تحليل الصورة من الرابط...
```

التحويل التلقائي يعني إنك ما تحتاج تبدّل النموذج يدوياً — تأسيس كود يختار نموذج الرؤية المناسب تلقائياً.

## ربط أي API

تأسيس كود يدعم الاتصال بأي خدمة متوافقة مع بروتوكول OpenAI — سواء Ollama على جهازك، LM Studio، أو أي خدمة أخرى.

### إضافة اتصال

```
❯ /api add ollama http://localhost:11434/v1
✔ تمت إضافة: ollama (http://localhost:11434/v1)

❯ /api add company-llm https://llm.company.sa/v1 sk-company-key-xxx
✔ تمت إضافة: company-llm (https://llm.company.sa/v1)
```

### إدارة الاتصالات

```
❯ /api list
الاتصالات المسجلة:
  ollama        http://localhost:11434/v1      بدون مفتاح
  company-llm   https://llm.company.sa/v1     مفتاح مسجل

❯ /api test ollama
✔ الاتصال ناجح: ollama (وقت الاستجابة: ٤٥ms)

❯ /api remove ollama
✔ تمت إزالة: ollama
```

- **Ollama** — تشغيل نماذج محلية على جهازك بدون إنترنت

## TASEESCODE.md — ذاكرة المشروع

أنشئ ملف `TASEESCODE.md` في جذر مشروعك وتأسيس كود يقرأه تلقائياً في كل مرة تبدأ فيها جلسة جديدة. هذا يعطيه سياق ثابت عن مشروعك.

### مثال

```markdown
# مشروعي

## نوع المشروع
تطبيق ويب Next.js 14 مع TypeScript

## هيكل المشروع
- src/app/ — صفحات App Router
- src/components/ — مكونات React
- src/lib/ — مكتبات مساعدة
- prisma/ — مخطط قاعدة البيانات

## القواعد
- استخدم TypeScript دائماً
- اكتب اختبارات لكل وظيفة جديدة
- التعليقات بالعربي
- اتبع هيكل App Router الجديد

## معلومات مهمة
- قاعدة البيانات: PostgreSQL عبر Prisma
- المصادقة: NextAuth.js
- النشر: Vercel
```

### ماذا يُكتب في الملف؟

- نوع المشروع والتقنيات المستخدمة
- هيكل المجلدات
- قواعد كتابة الكود
- معلومات البيئة والنشر
- أي شيء تريد أن يتذكره تأسيس كود عن مشروعك

## مرجع الإعدادات

الإعدادات محفوظة في `~/.taseescode/config.json`.

### الإعدادات الرئيسية

| المفتاح | الوصف | القيمة الافتراضية |
|---------|-------|-------------------|
| `language` | لغة الواجهة (`ar`, `en`, `auto`) | `auto` |
| `defaultModel` | النموذج الافتراضي | `deepseek-v3` |
| `theme` | مظهر الطرفية (`silver`, `minimal`, `dark`) | `silver` |
| `permissions.fileWrite` | السماح بكتابة الملفات | `true` |
| `permissions.commandRun` | السماح بتنفيذ الأوامر | `true` |

### مفاتيح API

| المفتاح | المزود |
|---------|--------|
| `apiKeys.deepseek` | DeepSeek |
| `apiKeys.anthropic` | Anthropic (Claude) |
| `apiKeys.openai` | OpenAI (GPT-4o) |
| `apiKeys.groq` | Groq (النماذج المجانية) |
| `apiKeys.qwen` | Qwen |
| `apiKeys.kimi` | Kimi |

### تعديل الإعدادات

```
❯ /config show
الإعدادات الحالية:
  language: auto
  defaultModel: deepseek-v3
  theme: silver
  ...

❯ /config set language ar
✔ تم تعيين language إلى: ar

❯ /config set defaultModel llama-3.3-70b
✔ تم تعيين defaultModel إلى: llama-3.3-70b
```

## دعم اللغة العربية

تأسيس كود صُمم ليكون عربي أولاً. هذا يعني:

### الكشف التلقائي عن اللغة

في وضع `auto`، تأسيس كود يكتشف لغة رسالتك ويرد بنفسها:

```
❯ اعمل لي دالة تتحقق من البريد الإلكتروني
✔ (يرد بالعربي)

❯ Create a function to validate email
✔ (responds in English)
```

### العربية في الكود

تقدر تكتب طلبك بالعربية السعودية العامية أو الفصحى:

```
❯ اعمل لي API بالنود
❯ سو لي كومبوننت React للتسجيل
❯ صلح الباق في ملف auth.ts
❯ أنشئ واجهة برمجة تطبيقات باستخدام Express
```

### لماذا العربية مهمة؟

المملكة العربية السعودية تعيش تحولاً تقنياً كبيراً ضمن رؤية ٢٠٣٠. آلاف المطورين السعوديين يدخلون سوق العمل كل سنة من خلال:

- **الاتحاد السعودي للأمن السيبراني والبرمجة (SAFCSP)**
- **أكاديمية طويق** — تخرّج آلاف المبرمجين سنوياً
- **برامج رؤية ٢٠٣٠** — التحول الرقمي في جميع القطاعات
- **سدايا** — الهيئة السعودية للبيانات والذكاء الاصطناعي
- **نموذج علّام** — نموذج KACST للذكاء الاصطناعي العربي

تأسيس كود موجود ليخدم هذا المجتمع المتنامي بأداة تفهم لغته واحتياجاته.

## تتبع التكلفة

تأسيس كود يتتبع تكلفة استخدامك بالريال السعودي (SAR).

```
❯ /cost
💰 تكلفة الجلسة الحالية:
  الإدخال:  ١٢,٤٥٠ رمز  →  ٠.٠٠٠٦٧ ر.س
  الإخراج:  ٣,٢١٠ رمز   →  ٠.٠٠٠٦٩ ر.س
  المجموع: ٠.٠٠١٣٦ ر.س

  النموذج: DeepSeek V3
```

### لماذا الريال السعودي؟

أغلب أدوات الذكاء الاصطناعي تعرض التكلفة بالدولار. تأسيس كود يعرضها بالريال لأن:

- المطور السعودي يحسب ميزانيته بالريال
- أسهل لمقارنة التكلفة مع خدمات أخرى
- شفافية كاملة في الأسعار

### تكلفة النماذج المجانية

النماذج عبر Groq (Llama, Qwen, ALLAM, Compound) **مجانية بالكامل**. ما يُحسب أي تكلفة عند استخدامها.

## استخدام متقدم

### تشغيل بدون معالج الإعداد

إذا عندك ملف إعدادات جاهز، تأسيس كود يتخطى المعالج تلقائياً:

```bash
# نسخ إعدادات جاهزة
cp my-config.json ~/.taseescode/config.json

# يبدأ مباشرة بدون معالج
taseescode
```

### استخدام مع Ollama (نماذج محلية)

```
❯ /api add ollama http://localhost:11434/v1
✔ تمت الإضافة

❯ /model ollama/llama3
✔ تم التبديل إلى النموذج المحلي
```

### استخدام في مشاريع الفريق

أنشئ ملف `TASEESCODE.md` في المشروع وارفعه مع Git. كل عضو في الفريق يحصل على نفس السياق:

```bash
echo "# قواعد المشروع\n- TypeScript فقط\n- اختبارات إلزامية" > TASEESCODE.md
git add TASEESCODE.md
git commit -m "إضافة ذاكرة تأسيس كود"
```

### تبديل سريع بين النماذج

```
❯ اشرح لي هالكود (يستخدم النموذج الافتراضي — سريع ورخيص)
❯ /model claude-sonnet
❯ راجع الكود هذا بدقة وابحث عن مشاكل أمنية (يستخدم Claude — أدق)
❯ /model llama-3.3-70b
❯ ايش هالخطأ؟ (يرجع للنموذج المجاني — سؤال بسيط)
```

## بناء مهارة

تقدر تبني مهارات مخصصة لتوسيع قدرات تأسيس كود.

### الخطوة ١: إنشاء مجلد المهارة

```bash
mkdir -p ~/.taseescode/skills/my-skill
```

### الخطوة ٢: كتابة skill.json

```json
{
  "name": "my-skill",
  "version": "1.0.0",
  "description": "وصف مهارتي المخصصة",
  "commands": ["/mycommand"],
  "systemPromptAddition": "أنت أيضاً خبير في [المجال]. عند استخدام /mycommand، قم بـ [السلوك المطلوب].",
  "author": "اسمك"
}
```

### الخطوة ٣: اختبار المهارة

```
❯ /skills list
المهارات المثبتة:
  my-skill    v1.0.0    /mycommand

❯ /mycommand ساعدني في...
✔ (يستخدم المهارة الجديدة)
```

### مشاركة المهارة

ارفع مجلد المهارة على GitHub وأي شخص يقدر يثبتها:

```
❯ /skills install username/my-skill
```

### أفكار لمهارات

- **saudi-api** — تكامل مع خدمات نفاذ، أبشر، مدى
- **flutter-ar** — تطوير تطبيقات Flutter مع دعم RTL
- **laravel-ar** — تطوير Laravel مع ترجمات عربية
- **react-native-ar** — تطبيقات React Native مع دعم العربي
- **aws-sa** — نشر على AWS في منطقة الخليج (me-south-1)
